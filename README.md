# node-storage-adapter
A storage-agnostic data adapter.

## API
### CRUD operations
* **#find(query, projection)**

## Pending
* **#insert(document|document array)**
* **#update(query, update, options)i**
* **#remove(query, options)**

### Aggregation
* **#aggregate(pipeline)**

### Pagination
* **#limit(count)**
* **#take(count)**

### Sorting
* **#sort(indeces)**
